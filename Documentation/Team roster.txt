PROJECT SCOPE AND OBJECTIVES:

Justification: CS Unplugged activities will help to educate people with computer science concepts. Searching and sorting concepts are important when we think about data. Building an interactive game about binary search will help to educate people with one of the searching concept.

Scope: This project will consist of a game about binary search. The modules of the game will include users looking for the cup under which the desired number is hidden, binary searching will help user to speed up the process and timer will help to check who did the best job.

Objectives:
-	Implement binary search logic
-	Implement User Interaction
-	Make multiple players able to play the game
-	Teach binary search concept using the game
